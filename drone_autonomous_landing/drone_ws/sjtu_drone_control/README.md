# package: sjtu_drone_control

간단 소개
- `sjtu_drone_control`은 드론 제어 알고리즘(예: 고수준 제어, PID 구성 등)과 ROS 인터페이스를 포함하는 패키지입니다.

사용법
- 패키지는 ament_python 또는 ament_cmake 구성을 따릅니다. 워크스페이스 루트에서 빌드하세요:
```
./scripts/build.sh
```

노트
- 패키지 설치/빌드 시 editable/install 관련 이슈를 막기 위해 `pyproject.toml`이 포함되어 있습니다.
